Semantic Debian

The Dataset

The Semantic Debian dataset currently contains details on debian releases, packaging projects, package releases, maintainer profiles and VCS repositories. It uses the ADMSSW, DOAP and FOAF RDF vocabularies to represent the data contained in the graph. An overview of the relations between these objects is shown below:

Debian Releases

URLs: http://rdf.debian.net/release/<release-name>

Packaging Projects

URLs: http://rdf.debian.net/project/<source-name>

Package Releases

URLs: http://rdf.debian.net/package/<source-name>_<debversion>

VCS Repositories

URLs: http://rdf.debian.net/repository/<source-name>

Maintainer Profiles

URLs: http://rdf.debian.net/maintainer/<url-encoded-email-address>
