Semantic Debian

Semantic Debian is a work-in-progress project to provide a Semantic Web frontend to Ultimate Debian Database. This interface will replace the deprecated interface provided by the old package tracking system.

The Dataset

The dataset is still being actively developed, and the URIs and vocabularies in use should be considered volatile. For more details on how the dataset is structured, see the dataset documentaion page.

The dataset currently contains data about:

  • Debian releases
  • Packaging projects
  • Package releases
  • Maintainer profiles
  • VCS repositories

Web Interface

You can browse the dataset online using the HTML interface. Content negotiation is available for all semantic resources in the dataset (unless it's currently broken - again, still being actively developed) and so you can also retrieve resource data as Turtle or RDF/XML. See the content negotiation examples for more details about using content negotiation.
