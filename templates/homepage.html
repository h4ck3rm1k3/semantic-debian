Semantic Debian

Semantic Debian is a work-in-progress project to provide a Semantic Web frontend to Ultimate Debian Database. This interface will replace the deprecated interface provided by the old package tracking system.

Initial goals:

  • Recreate the outputs provided by the old package tracker (#685605)
  • Look at integration with (or replacement of) webid.debian.net
  • RDF/XML, Turtle and HTML+RDFa outputs based on HTTP content negotiation
